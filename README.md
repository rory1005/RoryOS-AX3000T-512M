# RoryOS

> 一个基于 OpenWrt 的个人学习与定制项目。

RoryOS 目前以 **Xiaomi AX3000T V1** 为开发平台，
采用 **GitHub Actions** 自动编译，并逐步构建模块化开发框架。

本项目仍处于早期开发阶段，功能和目录结构会随着开发持续完善。

---

## 当前状态

**当前版本：** RoryOS 0.1 Alpha

目前已完成：

- GitHub Actions 自动编译
- Test-Only 开发模式
- 模块化脚本框架
- 自定义编译流程

正在开发：

- 系统基础设置
- 管理界面美化
- 软件包管理
- 无线网络配置
- AX3000T 专属优化
